# 🎮 ArcadeGames + Projetos Web I

Projeto desenvolvido durante a disciplina de Programação Web I utilizando HTML, CSS e JavaScript puro, sem frameworks.

O site reúne mini games estilo arcade e os quatro projetos avaliativos da disciplina, trabalhando conceitos como DOM, eventos, manipulação de classes, localStorage, timers e Web Audio API.

## 📚 Projetos Avaliativos

### 💡 Projeto 01 — Controle de Luz

#### Funcionalidades:
- Ligar e desligar lâmpada virtual  
- Mudança de ambiente junto com a luz  
- Contador de vezes acesa  
- Uso de `classList.toggle()`  
- Efeitos visuais e animações  

---

### 🐷 Projeto 02 — Cofrinho Digital

#### Funcionalidades:
- Adicionar moedas  
- Mostrar saldo atualizado  
- Saque com validação  
- Botão para esvaziar cofre  
- Salvamento com `localStorage`  
- Formatação com `Intl.NumberFormat`  

---

### ⏱ Projeto 03 — Pomodoro Timer

#### Funcionalidades:
- Timer regressivo  
- Iniciar, pausar e resetar  
- Configuração de tempo  
- Pausas curtas e longas  
- Sons ao finalizar  
- Uso de `setInterval()` e `clearInterval()`  

---

### 🎹 Projeto 04 — Teclado Musical

#### Funcionalidades:
- Piano virtual interativo  
- Sons gerados pela Web Audio API  
- Compatível com mouse e teclado físico  
- Gravação e reprodução de sequência  
- Efeitos visuais nas teclas
